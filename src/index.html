StencilJS Scoped Styles

The purpose of this demonstration is to understand how StencilJS applies scoped styles to Web Components.
In this demo we are:
  • Adding Bootstrap 4.x to demonstrate a whole framework included as "global" styles
  • Enabling general/global styles following Stencil's official mechanism (see /build/demo-stenciljs-scoped-styles.css)
  • Combining shadow: true with scoped: true
  • Consuming an actual REST API that serves a set of items (just showing one item for simplicity)
  • Using TypeScript for writing both Stencil components based on classes and functional components
  • Using utilities to format dates based on DayJS
  • Including Font Awesome 4 was included to demonstrate that even if global fonts are available with the Shadow DOM enabled, the styles for CSS classes fa fa-microphone aren't (a solution to this could be to create a web component that embeds the styles).
